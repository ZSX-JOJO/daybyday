  • 定位类position
    static position 的默认值，并没有对元素起到定位的作用,所以 left、right 等位置属性并不能被有效赋值，所以不存在在定位功能上的 % 值
    absolute %相对的元素便是这个参照物。left、right 是相对于 width，top 和 bottom 是相对于 height
    relative 子绝父相
    fixed %相对于视窗
  • 盒子模型
    width & height %相对于父盒子
    margin & padding %相对于父元素宽度,父元素宽度!父元素!!宽度!!!
    border 不存在%
    border-radius 前四个值和后四个值用 / 隔开，顺时针排序 斜杠前表示各角水平方向上的圆角半径，斜杠后表示各角竖直方向上的圆角半径
    border-radius:top-left|top-right|bottom-right|border-left / top-left|top-right|bottom-right|border-left
    border-radius 最多可以取八个值
  • 背景值
    background-posation 它的参照对象就是原盒子。但是这个属性比较特殊，他不是参照原盒子的宽高值，而是原盒子的宽高值减去背景图片的宽高值所得到的剩余值，更为形象的说，下面这两个属性值是等价的："center center" 和 "50% 50%"
  • transform
    translate
    translate(x,y) 参照物 自身的宽高
    translate3d(x,y,z) 当在z轴上进行偏移时 不起作用 因为z-index默认为auto
    transform-origin
    scale() 控制元素的缩放比例
    zoom 它的取值既可以是浮点数，也可以是 %，包括参照物，都与 scale() 等价
  • 字体
    font-size 参照父盒子
    line-height 如果属性值 是数字 最后的结果便是这个数字与该元素字体大小的乘积
    text-indent 缩进 如果设置的是 %，则参照的是父元素的 width 特例!!!
